# Grudy Terminal

A multi-tab terminal emulator with full functionality for modern development workflows.

## Features

- **Multi-tab Support**: Manage multiple terminal sessions simultaneously
- **Full Functionality**: Complete terminal capabilities for all your development needs
- **Lightweight & Fast**: Optimized performance for seamless user experience
- **Cross-platform**: Works across different operating systems

## Installation

```bash
npm install grudy-terminal
```

## Usage

```bash
grudy-terminal
```
